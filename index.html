Personal Game Project

I built this project in Java using the Eclipse IDE
for a networking class. Really I just wanted to get to program for my final project,
so I made this multiplayer game that implements a Java Server Socket. Then I uploaded it to the network VM
so I could play with other users on the VLAN!
It was my first forray in making a game, and probably the most impressive coding project I've made by myself.
There were a lot of questions without easy answers.
I also learned how useful git bash and bitbucket were for uploading a code project into a VM.
